# Windows系统查看用户登录等信息 #

## 一.打开“计算机管理”的三种方法 ##

1.实验操作系统：windows 7

2.方法一：

    ① 右键单击桌面“计算机”图标

    ② 选择“管理”选项

3.方法二：

    ① 打开计算机的“控制面板”

    ② 点击右上角“查看方式”，选择“大图标”

    ③ 选择“管理工具”

4.方法三：

    ① 组合键“Win+R”打开运行窗口

    ② 输入“compmgmt.msc”

## 二.查看系统用户登录日志 ##

1.选择“系统工具” → “事件查看器” → “Windows 日志” → “安全”
